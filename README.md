# truffle-compile-testing-migrtae(deploy)
## simply wrote two contracts and compiled, test and deployed(migrate) on the local network of truffle.

1.To create a folder in VS code. and install truffle 
command : npm install -g truffle

2.For required folders
command : truffle init

3.Write your contrat in contracts folder and to open the local truffle network
command: npx truffle develop

4.To compile your contract run the command and after compilation the json files will create in the build folder
command: compile

5.To test we have to write the script which is in test folder
command: test

6.After test now we can deploy our contract on the truffle test network
command: migrate

Done!!! your contract deployed :)
